NAME
    cd - decode the address into all supported character sets

SYNOPSIS
    cd {address}

DESCRIPTION
    The parameters are as follows:
    
    address     The address to look up in each character set. If prefixed
                with "0x" it will be read as a hexadecimal number. If
                prefixed with "0b" it will be read as a binary number. If
                there is no prefix, the UTF-8 address of the character
                will be used.
